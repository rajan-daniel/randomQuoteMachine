## 💬 Random Quote Machine (HTML, CSS & JavaScript Project)

### 📌 Preview  
A dynamic web app that generates random inspirational quotes and lets users instantly share them on Twitter using a built-in Twitter embed/share feature. Each new quote also triggers a visual theme change for a fresh user experience.

---

### 📖 About the Project  
The **Random Quote Machine** is an interactive web application that fetches random quotes from an external API and displays them in a clean, minimal interface. Each time the user generates a new quote, the app also changes the accent color dynamically, creating a fresh visual experience.

A key feature of this project is the **Twitter embed/share integration**, which automatically formats the current quote and author into a tweet-ready link using Twitter’s intent system. This allows users to share quotes instantly without leaving the app.

The project focuses on asynchronous data fetching, DOM manipulation, and enhancing user experience through simple but effective UI updates.

---

### 🚀 What I Learned  
- Working with **external APIs** using `fetch` and handling JSON responses  
- Implementing **asynchronous JavaScript** for dynamic content updates  
- Using **CSS variables** for real-time theming and color changes  
- Manipulating the **DOM to update UI elements instantly**  
- Creating **Twitter share (embed/intent) functionality** for social integration  
- Using **event listeners** to trigger UI and data changes  
- Improving UX with **randomized visual feedback (color changes)**
